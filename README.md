# Croesus Audio

Croesus Audio provides audio alerts for the first mechanics during gathering in 4man (usually until gathering for repairing statues has ended).

## Installing the app

Either:
- open https://audioalt1.github.io/croesus-audio/index.html in the alt1 browser or
- open `alt1://addapp/https://audioalt1.github.io/croesus-audio/appconfig.json`
